查询列名，返回多列：

```sql
SELECT COLUMN_NAME 
FROM `information_schema`.`COLUMNS` 
WHERE `TABLE_SCHEMA` = 'schema_name' AND TABLE_NAME = 'table_name';
```

查询列名，返回单列：

```sql
SELECT GROUP_CONCAT(COLUMN_NAME SEPARATOR ',') 
FROM `information_schema`.`COLUMNS` 
WHERE `TABLE_SCHEMA` = 'schema_name' AND TABLE_NAME = 'table_name';
```
